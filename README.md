# Hertz Fleet Utilization Analysis

## Project Overview
End-to-end SQL and Python analytics project analyzing vehicle fleet utilization,
pricing, and customer satisfaction using a public rental dataset.

## Business Questions
- Which fuel types drive the highest rental demand?
- Do alternative fuel vehicles receive higher customer ratings?
- How does pricing vary by vehicle type?

## Tools & Technologies
- Python (pandas)
- SQL
- Tableau (dashboard visualization)
- GitHub

## Key Insights
- Gasoline vehicles dominate overall rental demand
- Electric and hybrid vehicles receive the highest customer ratings
- Larger vehicle types (vans, SUVs) command higher daily rates
